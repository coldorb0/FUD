## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fireorb/FUD.git
   ```
2. Make to Execute
   ```sh
   cd FUD/ && chmod 777 fud.sh
   ```
4. Run
   ```sh
   sudo sh fud.sh
   ```
